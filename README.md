# 🌿 e-plantShopping – Paradise Nursery App

## 📌 Project Overview

e-plantShopping is a React-based shopping cart application for an online plant store called **Paradise Nursery**. The application allows users to browse a variety of houseplants, add them to a shopping cart, and manage their selections dynamically.

## 🚀 Features

* Landing page with company details and navigation
* Product listing page with categorized houseplants
* Add to cart functionality with dynamic updates
* Shopping cart page with:

  * Quantity increase/decrease
  * Item removal
  * Total cost calculation
* Responsive navigation bar with cart item count

## 🛠️ Technologies Used

* React (Function Components & Hooks)
* Redux Toolkit (State Management)
* JavaScript (ES6)
* CSS

## 📂 Repository Name

**e-plantShopping**

## 🌐 Deployment

The application is deployed using GitHub Pages.
